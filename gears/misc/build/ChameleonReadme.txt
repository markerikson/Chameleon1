Chameleon 1.0 - Readme

System Requirements:
Windows 9x or higher
Network connection
Available Linux server with SSH


Installation:
Run ChameleonSetup.exe, pick a directory to install to, and go.

To start Chameleon, double-click the Chameleon icon on the desktop, or select Chameleon from Start > Programs > Digital Gearheads.

See the Tutorial section of the help file for more details.
