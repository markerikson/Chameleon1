Menus

File

New File
Creates a new tab in the editor with a blank file.
Open File (Local)
Opens up a C++ source file for editing from the current computer.
Open File (Remote)
Opens up a C++ source file for editing from a remote server, using the settings specified in the Options dialog.
Close File
Closes the file in the active tab
Close All Files
Closes all open files
Save File
Saves the current active file. If the file has not been saved yet, this brings up a dialog asking whether to save the file on the current computer or on a remote server, then brings up a save file dialog. Otherwise, it simply saves the file.
Save File As (Local)
Brings up a save file dialog allowing the user to save the active file with a different name on the current computer.
Save File As (Remote)
Brings up a save file dialog allowing the user to save the active file with a different name on the remote server.
Print
Prints the current active file
Print Preview
Shows what the current active file will look like if it is printed
Print Setup
Lets the user change print-related settings such as margins and page size
Open Project File (Local)
Opens up a Chameleon project file off the current computer.
Open Project File (Remote)
Opens up a Chameleon project file off of a remote server, using the settings specified in the Options dialog.

Edit

Undo
Undoes the last action in the editor
Redo
Redoes the last undone action
Cut
Cuts out the highlighted text and places it on the Windows clipboard.
Copy
Copies the highlighted text and places it on the Windows clipboard.
Paste
Pastes the current contents of the Windows clipboard into the active editor file.
Indent text
Indents the highlighted text by one tab.
Unindent text
Unindents the highlighted text by one tab.
Select all
Selects all the text in the current file.
Select line
Selects all the text on the current line.

Compile

Compile [Project / File]
If there is no project open, compile the currently active file. If there is a project open, compile the project.

Project

Add file to project
Add a file to the currently open project.
Remove file from project
Removes a file from the currently open project.
Project settings
Modify the settings for the currently open project.

Debug

Run Program
Begins running the current program in debug mode. If the program has not been compiled yet, it will compile it automatically, then run it.
Stop Program
Stops the program that is currently running in debug mode.
Step next
Executes the next line of code if currently stopped at a breakpoint. If the next line is a function call, it will step into the function and execute the first line of the function.
Step over
Executes the next line of code if currently stopped at a breakpoint. If the next line is a function call, it will execute the function and stop at the next line of code in the current function.
Step out
Finishes executing the function the program is currently paused in, then stops at the first line of code after the current function.

Tools

Options
Open up the Chameleon Options dialog
Connect to server
Connect the terminal window to a remote server, using the settings specified in the Options dialog.
Disconnect from server
If the terminal is currently connected to a remote server, disconnect.

Help

Chameleon help contents
Opens up the Chameleon help file
About Chameleon
Opens up the About Chameleon dialog
