Version History

Version 1.1, February 2nd 2007

Changes
  • ADD: Added blank password check and password request dialog when opening remote files or trying to connect the terminal (should now only need to open the Options/Network page to change the username and hostname)
  • ADD: Added indefinite progress dialog to show something's happening during long network operations
  • ADD: Added ability to look at remote directories outside $HOME
  • ADD: Resizing the terminal window now informs the server of the resize
  • ADD: Line number printing is now optional
  • ADD: File icons in the RemoteFileDialog are now pulled directly from the user's system
  • CHG: Resizing the window now makes the terminal resize itself proportionally
  • CHG: SSH key caching has been improved
  • CHG: Removed references to MinGW (commented out until local compiling is added)
  • FIX: Disabled "Can't add image to imagelist" message while loading file icons
  • FIX: Fixed problem with debugging filenames that have an underscore
  • FIX: Fixed crash while loading non-source files (and disabled syntax highlighting for those files as well)
  • FIX: Fixed annoying repetitive "first chance exception" error caused by updating a non-existent toolbar
  • FIX: Fixed erratic terminal focus problems
  • FIX: Fixed crash caused that occurred when editing a previous commandline
  • FIX: Fixed problem when trying to view a directory the user doesn't have access to
Known Bugs
  • Print preview is still rather funky.
