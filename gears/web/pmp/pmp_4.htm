4.0 Technical Process

4.1 Methods, Tools, and Techniques

Our team plans to use the web-based application DotProject for project management, including task tracking, scheduling, Gantt Charts, and bug reports.  Coding tools will be left to each team member's discretion, but will likely include Microsoft Visual Studio under Windows and the Anjuta C++ IDE under Linux.  Code version updates will be handled using the CVS system, possibly using a web-based program such as CVSMonitor to show code status and progress.  The build process will be worked out as we begin writing code and near our first milestone.

4.2 Software Documentation

Documentation for the project will include the documents listed in section 1.4, the CVS commit logs, code comments, a user manual and tutorial, a maintenance manual, and in-code comments.
