6.2 Failing Test Cases

In this section, I want you to list the total number of test cases that failed, with respect to the total number of test cases (e.g. 7/30 test cases failed). Then list all the test cases that failed.

For every test case that failed, you must also explain why the test failed (e.g. function not implemented, implementation error, etc.)
