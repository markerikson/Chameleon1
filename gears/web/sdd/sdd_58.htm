5.8 User Interfaces

5.8.1 Introduction

Provide an introductory paragraph describing in general the user interface design.
--JC.
The user interface design is similar to most standard IDEs: menubar and toolbar on top, editing window in the middle, and some form of information window in the bottom of the screen.  It uses standard Windows GUI components and should have a normal "Windows application" appearance.
