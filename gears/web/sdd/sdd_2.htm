2.0 Development and Execution Environment

2.1 Development Environment

2.1.1 Hardware

Describe the hardware that will be used in the development environment. For example, you will probably list all the PCs and related hardware that you and your team will be using to develop this product.
--JC.
Chameleon is being developed on standard x86-based computers.

2.1.2 Software

Development will be done under Windows using a variety of software tools.  These may include Microsoft Visual Studio versions 6 and 7 and their related C++ compilers; the vi editor and the mingw C++ compiler; the CVS server and various CVS clients, including TortoiseCVS; and various other utilities as necessary.

2.2 Execution Environment

2.2.1 Hardware

Chameleon will run on standard x86-based computers.  It requires a network connection for remote access.  Hardware problems are not likely to be an issue.

2.2.2 Software

Chameleon should run under any 32-bit Windows operating system, from Windows 95 to Windows XP.
