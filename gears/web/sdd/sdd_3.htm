3.0 Naming and Coding Standards

3.1 Naming Standards

Our naming standards should follow standards set by the Cedarville Computer Science Dept., The C++ Style Guide. No deviation from this style guide is expected, and no additions are necessary.

3.2 Coding Standards

Our coding standards should follow standards set by the Cedarville Computer Science Dept., The C++ Style Guide. No deviation from this style guide is expected. Additionally we will be emphasizing liberal use of wxDebug for tracking and logging debug information throughout the code.
