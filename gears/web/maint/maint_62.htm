6.2 - Source Files

  • /
    • Chameleon.rc - resource file (defines the application icon)
    • chameleon.ico - the application icon
  • common/
    • ChameleonPrintout.cpp/.h - provides required printing functionality
    • chameleonprocessevent.cpp/.h - custom event class used by Networking
    • CommonHeaders.h - Frequent #include's are listed here
    • Crc16.cpp/.h - Checksum routines used to verify authorization codes
    • datastructures.h - Program-wide enumerations and datastructures
    • debug.h - enables memory leak checks
    • DebugEvent.cpp/.h - custom event class used by ChameleonWindow and Debugger
    • options.cpp/.h - Stores program options, used to initialize the state of the options dialog
    • ProjectInfo.cpp/.h - Stores information relating to a project, whether single- or multi-file
    • fixvsbug.h - This is a fix for the VisualStudio 6.0 for-loop scoping bug
  • compiler/
    • compiler.cpp/.h - Compiler class
    • compilerevent.cpp/.h - custom event used by Compiler and ChameleonWindow
  • debugger/
    • cham_db.cpp/.h - Debugger class
  • editor/
    • editor.cpp/.h - Editor class
  • gui/
    • dialogs/
      • AboutChameleon.pjd - DialogBlocks project file for the about box
      • AboutChameleonDialog.cpp/.h - About box
      • AddVariableWatch.cpp/.h - Dialog for adding variable watches (wound up unused, but might be useful later)
      • AddVariableWatch.pjd - DialogBlocks project file for the AVW dialog
      • CompilerOutputPanel.cpp/.h - Displays compiler output in a textbox or grid
      • CompilerOutputPanel.pjd - DialogBlocks project file for the COP
      • opensave.pjd - DialogBlocks project file for the RemoteFileDialog
      • OptionsDialog.cpp/.h - The options dialog
      • options.pjd - DialogBlocks project file for the
      • RemoteFileDialog.cpp/.h - Custom file dialog for showing remote files, modeled after the Windows common dialog
      • VariableWatchPanel.cpp/.h - Displays the values of variables while debugging
      • VariableWatchPanel.pjd - DialogBlocks project file for the VWP
      • wxTermContainer.cpp/.h - Holds a wxTerm widget and a scrollbar
      • wxTermContainer.pjd - DialogBlocks project file for the wxTermContainer
      • *.xpm - Icons used in the various dialogs
    • Chameleon.cpp - Contains the guts of the program, including the main application class ChameleonWindow
    • ChameleonNotebook.cpp/.h - Extra custom functionality on top of the standard notebook widget
    • ChameleonWindow.h - Declaration of the ChameleonWindow class
    • wxssh.cpp/.h - Derivation of wxTerm, specialized for SSH I/O
    • wxterm.cpp/.h - VT100-capable terminal widget
    • *.xpm - Icons used in the various windows
  • network/
    • actions.cpp - GTerm state machine actions
    • gtelnet.cpp/.h - GTelnet class definition
    • gterm.cpp/.h - GTerm class, the abstract core behind wxTerm
    • keytrans.cpp - GTerm keycode stuff
    • networking.cpp/.h - Networking class definition
    • plinkconnect.cpp/.h - PlinkConnect class definition
    • states.cpp - Telnet (vt100) state definitions used by GTerm
    • TextManager.cpp/.h - Handles terminal history and scrolling
    • utils.cpp - GTerm's main functions
    • vt52_states.cpp - Telnet (vt52) state definitions used by GTerm
    • winmesgbeep.cpp - GTerm bell definition
  • perms/
    • p.cpp/.h - Permissions class
  • profmanager/
    • OptionsManager.pjd - DialogBlocks project file for the Professor Options program
    • ProfOptionsDialog.cpp/.h - The dialog for the Professor Options program
    • ProfOptionsManager.cpp - Application code for the Professor Options program
