A2 Professor Program

Description:

The "Professor Program" is the extension that lets the professor of a computer-science course customize Chameleon to the perceived / actual level of competency of the students. It's a small program with a list of all the features in Chameleon with the option to activate ("authorize") or deactivate each individual one.

Purpose:

The purpose of this program was the core purpose behind the Chameleon project: to hide the excess complications of an advanced IDE to students who are not ready for an advanced IDE. It works in conjunction with the Permissions Module to hide or show items that are considered "advanced". This program can reduce Chameleon to a simple text editor, or raise it to a full IDE complete with debugging, compiling, network-ready file saving, and project management.
