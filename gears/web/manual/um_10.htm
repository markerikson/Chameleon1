1.0 Introduction

Welcome to Chameleon, the Adaptive Integrated Development Environment.

1.1 What is Chameleon?

Chameleon is an Itegrate Development Environment (IDE for short). It is a single program that has consolidated programming tools and functions into one unified environment to assist in code development.
